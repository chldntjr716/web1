html

hypertext markup language (html) is th standard markup language for creating web pages and web applications. web browsers receive html documents from a web server or from local storage and render them into multimedia web pages. html describes the structure of a web page semantically and originally includes cues for the appearance of the document.

html elements are the building blocks of html pages. with html constructs, images and other objects, such as intractive forms, may be embedded into the redered page. it provides a means to create structured documents by denoting structural semantics for text such as headings, paragraphs, lists, links, quotes and other items. html elements are delineated by tags, written using angle brackets.
